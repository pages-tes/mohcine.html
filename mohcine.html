Comment créer un tableau en HTML

Comment créer un tableau en HTML

1. Structure du tableau

Pour créer un tableau en HTML, vous pouvez utiliser les éléments suivants :

  • <table> : pour définir le début du tableau.
  • <tr> : pour créer une ligne dans le tableau.
  • <th> : pour créer une cellule d'en-tête dans la ligne.
  • <td> : pour créer une cellule de données dans la ligne.

2. Mise en forme du tableau

Vous pouvez utiliser CSS pour styliser le tableau et le rendre plus attrayant. Par exemple :


        table {
            width: 100%;
            border-collapse: collapse;
        }
        th, td {
            border: 1px solid #ddd;
            padding: 8px;
            text-align: left;
        }
        th {
            background-color: #f2f2f2;
        }

Explication des éléments de formulaire en HTML

1. Balise <form>

La balise <form> est utilisée pour créer un formulaire pour collecter des données utilisateur. On peut spécifier des propriétés comme action (la destination) et method (la méthode d'envoi).

2. Champs de saisie (Input, TextArea)

La balise <input> est utilisée pour créer des champs de saisie, tandis que <textarea> permet de saisir du texte sur plusieurs lignes.

<input type="text" name="username">
<textarea rows="4" cols="50">Écrivez ici...</textarea>

3. Balises de choix (RadioButton, CheckBox, Select)

Les éléments comme <input type="radio"> et <input type="checkbox"> permettent à l'utilisateur de choisir des options, tandis que <select> est utilisé pour créer une liste déroulante.

4. Balises HTML5 pour les formulaires

HTML5 a ajouté de nouveaux éléments comme <datalist> et <output> pour améliorer l'interaction avec les formulaires.

5. Validation automatique

HTML permet la validation automatique des données saisies grâce aux propriétés intégrées telles que required et pattern.

6. Boutons

On peut utiliser la balise <button> pour créer des boutons, avec des options pour définir le type de bouton comme soumission ou réinitialisation.

<button type="submit">Soumettre</button>

Introduction au CSS

1. Présentation du CSS

Le CSS (Cascading Style Sheets) est un langage utilisé pour décrire la présentation d'un document écrit en HTML. Il permet de contrôler la mise en page, les couleurs, les polices et d'autres aspects visuels d'une page web.

2. Codification des couleurs

Les couleurs en CSS peuvent être définies de plusieurs manières :

  • Nom de couleur (ex. : red)
  • Valeurs hexadécimales (ex. : #ff0000)
  • Valeurs RGB (ex. : rgb(255, 0, 0))
  • Valeurs RGBA (ajout de transparence, ex. : rgba(255, 0, 0, 0.5))

3. Unités de mesure

Les unités de mesure en CSS incluent :

  • px : pixels
  • em : taille de police relative
  • rem : taille de police relative à l'élément racine
  • % : pourcentage

4. Positions (center, left, right)

Le positionnement des éléments peut être contrôlé par des propriétés telles que :

  • text-align : pour centrer, aligner à gauche ou à droite le texte.
  • margin : pour centrer un élément en utilisant des marges automatiques.

5. Fonts

Les polices en CSS peuvent être définies avec des propriétés telles que :

  • font-family : pour spécifier la police.
  • font-size : pour définir la taille de la police.
  • font-weight : pour définir le poids (normal, bold, etc.).

6. Types d’intégration du CSS

Il existe trois façons d'intégrer le CSS dans un document HTML :

  • CSS en ligne : <h1 style="color:red;">Titre</h1>
  • CSS interne : en utilisant une balise <style> dans l'en-tête.
  • CSS externe : en liant un fichier CSS externe avec <link>.

7. Sélecteurs simples

Les sélecteurs simples permettent de cibler des éléments spécifiques dans un document HTML :

  • Sélecteur d'élément : cible tous les éléments d'un type donné. Exemple : p pour tous les paragraphes.
  • Sélecteur de classe : cible tous les éléments avec une classe spécifique. Exemple : .ma-classe pour tous les éléments ayant cette classe.
  • Sélecteur d'ID : cible un élément unique avec un ID spécifique. Exemple : #mon-id pour un élément ayant cet ID.

8. Sélecteurs complexes

Les sélecteurs complexes permettent de combiner plusieurs sélecteurs simples :

  • Descendants : div p cible tous les p à l'intérieur d'un div.
  • Enfants : ul > li cible tous les li qui sont enfants directs d'un ul.
  • Adjacents : h1 + p cible le premier p qui suit immédiatement un h1.

9. Pseudo classes

Les pseudo-classes permettent de cibler des éléments dans des états spécifiques :

  • :hover : s'applique lorsque l'utilisateur survole un élément. Exemple : a:hover change la couleur du lien au survol.
  • :first-child : cible le premier enfant d'un parent. Exemple : li:first-child cible le premier élément de liste.
  • :nth-child(n) : cible l'enfant en fonction de sa position. Exemple : tr:nth-child(2) cible le deuxième rang d'un tableau.
